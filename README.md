# 🍽️ Swiggy Sales Analysis Dashboard

## 📌 Overview

This project showcases an interactive **Microsoft Excel dashboard** developed to analyze Swiggy sales data. The dashboard transforms raw transactional data into meaningful business insights through interactive visualizations, KPIs, and filters.

It helps analyze sales trends, customer behavior, regional performance, and order patterns to support data-driven decision making.

---

## 🎯 Objectives

* Analyze overall sales performance
* Track daily, weekly, and monthly sales trends
* Compare sales across different states and cities
* Identify top-performing food categories
* Monitor customer ratings and order volume
* Build an interactive dashboard using Microsoft Excel

---

## 📊 Dashboard KPIs

* **Total Sales:** ₹53.01M
* **Average Rating:** 4.34
* **Average Order Value:** ₹268.51
* **Total Orders:** 197.43K

---

## 📈 Dashboard Features

* KPI Cards
* Daily, Weekly & Monthly Sales Trends
* State-wise Sales Analysis
* Top 5 Cities by Sales
* Food Category Analysis
* Quarterly Sales, Ratings & Orders
* Interactive Slicers and Filters

---

## 💡 Key Insights

* Saturday recorded the highest sales among all weekdays.
* Bengaluru generated the highest sales.
* Non-Veg orders contributed the largest share of total sales.
* Customer ratings remained consistently above 4.3 throughout the year.
* Interactive filters make it easy to explore sales across different dimensions.
